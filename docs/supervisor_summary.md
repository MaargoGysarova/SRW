# Короткое объяснение руководителю

В НИРС я строю базовый конвейер для выявления мошенничества в русскоязычных диалогах с помощью LLM-подхода. Основной эксперимент проводится на контролируемом синтетическом текстовом датасете, потому что открытых русскоязычных датасетов такого типа практически нет.

В рамках работы я:

- формирую синтетический датасет `fraud / suspicious / safe`;
- задаю таксономию сценариев и список сигналов мошенничества;
- сравниваю несколько архитектур: правила, `single_llm`, `llm_checklist`, `llm_self_check` и `llm_ensemble`;
- отдельно проверяю устойчивость к переформулировкам и ASR-подобному шуму;
- использую аудиочасть как дополнительную схему обработки: аудио, распознавание речи и последующая классификация текста.

Таким образом, работа оформляется как исследовательский базовый проект с воспроизводимым кодом, метриками, ограничениями и понятным переходом к дипломной работе.
